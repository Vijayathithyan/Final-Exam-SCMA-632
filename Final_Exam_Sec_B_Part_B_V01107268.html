# Co-Integration Test (Johansen's Test)
def johansen_test(df, alpha=0.05):
    out = coint_johansen(df, det_order=0, k_ar_diff=1)
    d = {'0.90': 0, '0.95': 1, '0.99': 2}
    traces = out.lr1
    cvts = out.cvt[:, d[str(1 - alpha)]]
    print(f"Trace statistic: {traces}")
    print(f"Critical values: {cvts}")
    print(f"Eigenvalues: {out.eig}")
    for col, trace, cvt in zip(df.columns, traces, cvts):
        if trace > cvt:
            print(f"{col} is cointegrated.")
        else:
            print(f"{col} is not cointegrated.")
    return out

# Prepare data for LSTM
def create_dataset(data, look_back=1):
    X, Y = [], []
    for i in range(len(data) - look_back):
        X.append(data[i:(i + look_back)])
        Y.append(data[i + look_back])
    return np.array(X), np.array(Y)

Interpretation of Results¶

Root Mean Squared Error (RMSE): This metric measures the square root of the average squared differences between predicted and actual values. Lower values indicate better performance.¶
The RMSE for the LSTM model (88.891) is significantly lower than that for the SARIMA model (375.306), suggesting that the LSTM model provides more accurate predictions.¶
Mean Absolute Percentage Error (MAPE): This metric measures the average absolute percentage error between predicted and actual values. Lower percentages indicate better performance.¶
The LSTM model has a MAPE of 4.724%, much lower than the SARIMA model's 23.376%. This indicates that the LSTM model's predictions are closer to the actual values in terms of percentage error.¶
Mean Absolute Error (MAE): This metric measures the average absolute differences between predicted and actual values. Lower values indicate better performance.¶
The MAE for the LSTM model (71.347) is considerably lower than that for the SARIMA model (308.754), again indicating that the LSTM model performs better in terms of average error.¶

Best-Fit Model¶

Based on the evaluation metrics RMSE, MAPE, and MAE, the LSTM model is the best-fit model for the data. It outperforms the SARIMA model across all metrics, providing more accurate and reliable predictions.¶
